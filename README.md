# Ethics and Bias Audit in Data Science

## Project Overview
This project evaluates fairness and bias in a machine learning hiring model.

The goal was to audit model predictions across gender groups and identify ethical concerns related to fairness in automated decision-making.

## Objectives
- Build a hiring prediction model
- Evaluate model performance
- Audit fairness across gender groups
- Detect potential bias in predictions

## Skills Demonstrated
- Python
- Pandas
- NumPy
- Scikit-learn
- Fairness Metrics
- Bias Detection
- Ethical AI Analysis

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn

## Key Findings
- Overall Accuracy: 77%
- Male Accuracy: 75.97%
- Female Accuracy: 78.08%

### Positive Prediction Rate
- Male: 42.21%
- Female: 41.10%

### False Positive Rate
- Male: 0.00%
- Female: 27.12%

### False Negative Rate
- Male: 36.27%
- Female: 0.00%

## Ethical Concerns
Although overall performance was balanced, the model showed unequal error rates across gender groups, indicating fairness concerns.

## Recommendations
- Audit training data regularly
- Apply fairness-aware machine learning techniques
- Monitor fairness metrics continuously
- Include human oversight in high-stakes decisions

## Conclusion
Machine learning models should be both accurate and fair. Bias audits help ensure ethical and responsible AI systems.
